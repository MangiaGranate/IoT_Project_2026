
# Progetto IoT


## Proposte:

- `Monitoraggio di consumi anomali` / `Controllo di macchinari industriali energivori`

- `Rilevamento perdite d’acqua in un condominio`

- `Gestione intelligente di serre o orti urbani`
 
- `Rilevamento vibrazioni per manutenzione predittiva`


## Consumi anomali di energia elettrica o Controllo macchinari
```
Orientato a macchinari industriali ad alto consumo.  
Per monitoraggio di dispositivi critici.  
```


### *Sensori/Attuatori*:

- **Smart plug** (per prototipi domestici)

- **Analizzatore di rete e relè** (in ambito industriale)



## Rilevamento perdite d’acqua
```
Molto simile concettualmente al primo caso, ma specifico per acqua.  
```


### *Sensori/Attuatori*:

- **Contatore**

- **Elettrovalvole** 


## Gestione intelligente di serre o orti urbani
```
Raccolta dati e attuazione per gestione serra.
```

### *Sensori/Attuatori*:

- **Sensore umidità**

- **Irrigazione**

- **Sistemi di ventilazione**

- **Sensore di CO2**


## Rilevamento vibrazioni per manutenzione predittiva
```
Raccolta dati per predirre guasto di un dispositivo critico.
```


### *Sensori/Attuatori*:

- **Sensore vibrazione**

- **Sensore temperatura**

- **Ventole (gestore temperatura)**
